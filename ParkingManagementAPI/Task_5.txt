────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
RELATIONSHIPS + MODEL ENTITIES
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

ParkingLot
│
└── Floor (1:N Parking Lot)
   │
   └── ParkingSpot (1:N Floor, 1:N Parking Ticket) - Abstract Base Class - Inherited By:
       ├── CompactSpot - AcceptableVehicle(vehicle is Motorcycle || vehicle is Car)
       ├── LargeSpot - AcceptableVehicle(vehicle is Truck)
       └── HandicappedSpot - AcceptableVehicle(vehicle is Motorcycle || vehicle is Car && vehicle.PassengerIsHandicapped)

Every ParkingLot:
├── int ParkingLotId (Surrogate PK)
├── string ParkingLotName
├── string Address
└── ICollection<Floor> Floors

Every Floor:
├── int FloorId (PK)
├── int FloorNumber
├── int ParkingLotId (FK)
├── ParkingLot ParkingLot (Nav Property)
└── ICollection<ParkingSpot> ParkingSpots 

Every ParkingSpot:
├── int ParkingSpotId (PK)
├── string ParkingSpotName (E.g. A01)
├── int FloorId (FK)
├── Floor Floor (Nav Property)
├── bool IsOccupied (private set)
├── ICollection<ParkingTicket> TicketHistory (ICollection<ParkingTicket>)
├── void OccupySpot(IsOccupied = true)
├── void FreeSpot(IsOccupied = false)
└── abstract bool AcceptableVehicle()

      ──────────────────────────────────────────────────────────────────────────────────

Vehicle (1:N Parking Ticket) - Abstract Base Class - Inherited By:
├── Motorcycle
├── Car
└── Truck

Every Vehicle:
├── int VehicleId (PK)
├── string LicensePlate
├── string Make
├── string Model
├── bool PassengerIsHandicapped
└── ICollection<ParkingTicket> TicketHistory

      ──────────────────────────────────────────────────────────────────────────────────

ParkingTicket (Joining Table for Vehicle + Parking Spot)
├── int ParkingTicketId (PK)
├── int VehicleId (FK)
├── Vehicle Vehicle (Nav Property)
├── int ParkingSpotId (FK)
├── ParkingSpot ParkingSpot (Nav Property)
├── DateTime TimeOfIssuance
├── DateTime? TimeOfConclusion? (Nullable because could still be in progress)
├── decimal? TotalFee? (Nullable because could still be in progress)
├── FeeStrategyType (Hourly, Daily, Flat as Enum)
└── bool IsOpen => TimeOfConclusion == null

────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
DTOS + MAPPER
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

ParkingTicketResponseDTO - { get; init; } on all fields instead of { get; set; }
├── int TicketId
├── int FloorNumber
├── string ParkingSpotName
├── DateTime TimeOfIssuance
├── DateTime? TimeOfConclusion
├── decimal? TotalFee
├── FeeStrategyType
└── bool IsOpen

ParkVehicleRequestDTO - references Enum VehicleType { Motorcycle, Car, Truck }
├── string LicensePlate [Required]
├── string Make
├── string Model
├── bool PassengerIsHandicapped [Required]
├── VehicleType VehicleType [Required]
└── FeeStrategyType [Required]

ParkingMapper
│
├── ToVehicle(ParkVehicleRequestDTO): return dto.VehicleType switch
│   │
│   ├── VehicleType.Car
│   │   └── Car
│   │       ├── LicensePlate ← DTO.LicensePlate
│   │       ├── Make ← DTO.Make
│   │       ├── Model ← DTO.Model
│   │       └── PassengerIsHandicapped ← DTO.PassengerIsHandicapped
│   │
│   ├── VehicleType.Motorcycle
│   │   └── Motorcycle
│   │       ├── LicensePlate ← DTO.LicensePlate
│   │       ├── Make ← DTO.Make
│   │       ├── Model ← DTO.Model
│   │       └── PassengerIsHandicapped ← DTO.PassengerIsHandicapped
│   │
│   ├── VehicleType.Truck
│   │   └── Truck
│   │       ├── LicensePlate ← DTO.LicensePlate
│   │       ├── Make ← DTO.Make
│   │       ├── Model ← DTO.Model
│   │       └── PassengerIsHandicapped ← DTO.PassengerIsHandicapped
│   │
│   └── Invalid VehicleType
│       └── InvalidVehicleTypeException
│
└── ToResponse(ParkingTicket)
    │
    └── return new ParkingTicketResponseDTO
        ├── TicketId ← ParkingTicket.ParkingTicketId
        ├── FloorNumber ← ParkingSpot.Floor.FloorNumber
        ├── ParkingSpotName ← ParkingSpot.ParkingSpotName
        ├── TimeOfIssuance ← ParkingTicket.TimeOfIssuance
        ├── TimeOfConclusion ← ParkingTicket.TimeOfConclusion
        ├── TotalFee ← ParkingTicket.TotalFee
        ├── FeeStrategyType ← ParkingTicket.FeeStrategyType
        └── IsOpen ← ParkingTicket.IsOpen
